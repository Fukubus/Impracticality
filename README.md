## Features
- Flesh Cubes that can and will decay over time.
- Ground variants of the above.
- Deepslate tools.
- Blackstone tools.
- Portable crafting table.
### Vanilla Changes
- Stone tools can no longer be made/repaired with deepslate or blackstone.
